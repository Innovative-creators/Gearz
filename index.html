Q: When I type the help command the bot shows more commands, but when a user in my server types the help command the bot shows less commands, why?
A: Its becuase the bot is coded in such a way that, you will only see the commands which you have permission to either in a guild or DM. Also a user will be able to use only those commands which bot replied when he/she use the help command.

Q: The bot has every permission in the guild, but it gives me some strange replies as {user}, You cannot Kick {someone}, why is it happening?
A: Ah its not the issue with the bot, but instead some issue with discord itself. To make this not happen you will have to move the role of the bot i.e. Gearz to a higher position in the role list, and better to put it at the top to avoid any futher similar issues.

Q: How do I set welcome-goodbye messages for the bot in my server?
A: Do {prefix}set -edit userLog true
Do {prefix}set -edit logChannel channelid
Do {prefix}set edit userLogType (1/2/3)
Types:
1. Simple text
2. Embed text
3. Image
